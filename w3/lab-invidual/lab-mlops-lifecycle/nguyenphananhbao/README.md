# MLOps Lifecycle Pipeline — Nguyễn Phan Anh Bảo

Để chạy toàn bộ hệ thống từ đầu đến cuối, trước tiên hãy khởi động hạ tầng bằng lệnh `bash data-pack/scripts/start_stack.sh`, sau đó sinh dữ liệu bằng `uv run python data-pack/data/generate_data.py`. Tiếp theo, huấn luyện model v1 bằng lệnh `uv run python nguyenphananhbao/pipeline.py --data data-pack/data/baseline.csv` và khởi chạy server API bằng `uv run python nguyenphananhbao/serve.py`. Cuối cùng, chạy orchestrator bằng lệnh `uv run python nguyenphananhbao/retrain.py --reference data-pack/data/baseline.csv --current data-pack/data/drifted.csv --holdout data-pack/data/holdout.csv --post-deploy-eval data-pack/data/post_deploy_eval.csv --auto-approve` để tự động phát hiện trôi lệch dữ liệu, huấn luyện model v2, phê duyệt, triển khai reload không downtime và giám sát kích hoạt cơ chế auto-rollback.
